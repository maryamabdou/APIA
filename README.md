# APIA

## Run the website

### Frontend

Go to client folder

```bash
  cd client/
```

Install Dependencies

```bash
  npm install --force
```

Start the server

```bash
  npm start
```

### Backend

Go to flask-server folder

```bash
  cd flask-server/
```

Install Dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python server.py
```
